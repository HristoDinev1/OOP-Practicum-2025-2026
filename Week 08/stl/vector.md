# `std::vector` в C++

`std::vector` е динамичен масив от стандартната библиотека на C++. За разлика от обикновените масиви, векторът може да променя размера си по време на изпълнение на програмата.

```cpp
#include <vector>
using namespace std;
```

---

## Създаване (Инициализация)

```cpp
vector<int> v1;                        // празен вектор
vector<int> v2(5);                     // 5 елемента, всички = 0
vector<int> v3(5, 42);                 // 5 елемента, всички = 42
vector<int> v4 = {1, 2, 3, 4, 5};      // инициализация със списък
vector<int> v5(v4);                    // копие на v4
```

---

## Добавяне на елементи

### `push_back(val)` — добавя елемент в края

`push_back` приема **вече създаден обект** и го **копира или мести** във вектора.
Това означава, че обектът се създава веднъж, и след това се прави
**допълнителна операция** (копиране/местене) за да влезе във вектора.

```cpp
vector v;
v.push_back(10);   // int се копира директно — няма разлика при прости типове
v.push_back(20);
// v = {10, 20}
```

При по-сложни типове разликата става видима:

```cpp
vector v;

string s = "Hello";
v.push_back(s);         // 1. s е вече създаден, 2. копира се във вектора
v.push_back("World");    // 1. създава се временен string, 2. местви се във вектора
```

---

### `emplace_back(args...)` — конструира директно в края

`emplace_back` **не приема готов обект** — приема **аргументите на конструктора**
и създава обекта **директно на място** в паметта на вектора.
Няма копиране, няма местене. Само едно единствено създаване.

```cpp
vector v;
v.emplace_back("Hello");  // string се конструира директно вътре във вектора
```


### `insert(pos, val)` — вмъква на дадена позиция
```cpp
vector<int> v = {1, 2, 4, 5};
v.insert(v.begin() + 2, 3);  // вмъква 3 на позиция 2
// v = {1, 2, 3, 4, 5}
```

## Премахване на елементи

### `pop_back()` — премахва последния елемент
```cpp
vector<int> v = {1, 2, 3};
v.pop_back();
// v = {1, 2}
```

### `erase(pos)` — премахва елемент на дадена позиция
```cpp
vector<int> v = {10, 20, 30, 40};
v.erase(v.begin() + 1);  // премахва елемента на индекс 1
// v = {10, 30, 40}
```

### `erase(start, end)` — премахва диапазон от елементи
```cpp
vector<int> v = {1, 2, 3, 4, 5};
v.erase(v.begin() + 1, v.begin() + 3);  // премахва индекси 1 и 2
// v = {1, 4, 5}
```

### `clear()` — премахва всички елементи
```cpp
vector<int> v = {1, 2, 3};
v.clear();
// v = {} , v.size() == 0
```

---

## Достъп до елементи

### `v[i]` — достъп по индекс (без проверка на граници!)
```cpp
vector<int> v = {10, 20, 30};
cout << v[0];  // 10
cout << v[2];  // 30
```

### `v.at(i)` — достъп по индекс (с проверка, хвърля изключение при грешен индекс)
```cpp
vector<int> v = {10, 20, 30};
cout << v.at(1);   // 20
cout << v.at(10);  // хвърля std::out_of_range
```

### `front()` и `back()` — първи и последен елемент
```cpp
vector<int> v = {5, 10, 15};
cout << v.front();  // 5
cout << v.back();   // 15
```

## Размер и капацитет

### `size()` — брой елементи
```cpp
vector<int> v = {1, 2, 3, 4};
cout << v.size();  // 4
```

### `empty()` — проверява дали векторът е празен
```cpp
vector<int> v;
if (v.empty()) {
    cout << "Векторът е празен!";
}
```

### `capacity()` — колко памет е заделена (може да е повече от size)
```cpp
vector<int> v;
v.push_back(1);
cout << v.size();      // 1
cout << v.capacity();  // обикновено 1 или повече (зависи от компилатора)
```

### `reserve(n)` — заделя памет предварително (оптимизация)
```cpp
vector<int> v;
v.reserve(100);  // заделя място за 100 елемента без да ги добавя
cout << v.size();      // 0
cout << v.capacity();  // 100
```

### `resize(n)` — променя размера на вектора
```cpp
vector<int> v = {1, 2, 3};
v.resize(5);       // добавя 2 нула-елемента
// v = {1, 2, 3, 0, 0}
v.resize(2);       // отрязва до 2 елемента
// v = {1, 2}
```

## Итерация (обхождане)

### С `for` цикъл и индекс
```cpp
vector<int> v = {10, 20, 30};
for (int i = 0; i < v.size(); i++) {
    cout << v[i] << " ";
}
// 10 20 30
```

### С range-based `for` (препоръчително)
```cpp
vector<int> v = {10, 20, 30};
for (int x : v) {
    cout << x << " ";
}
// 10 20 30
```

### С итератори
```cpp
vector<int> v = {10, 20, 30};
for (auto it = v.begin(); it != v.end(); ++it) {
    cout << *it << " ";
}
// 10 20 30
```

## Сортиране и търсене

### `sort()` — сортиране (от `<algorithm>`)
```cpp
#include <algorithm>

vector<int> v = {5, 2, 8, 1, 9};
sort(v.begin(), v.end());
// v = {1, 2, 5, 8, 9}

sort(v.begin(), v.end(), greater<int>());  // низходящ ред
// v = {9, 8, 5, 2, 1}
```

### `find()` — търсене на елемент
```cpp
#include <algorithm>

vector<int> v = {3, 7, 1, 9};
auto it = find(v.begin(), v.end(), 7);

if (it != v.end()) {
    cout << "Намерен на позиция: " << (it - v.begin());  // 1
} else {
    cout << "Не е намерен";
}
```
